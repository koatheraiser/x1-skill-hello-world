Basic greeting skill for X1 (based off MycroftAI system)

## About
This is a basic Greeting / Thank you Skill that takes an _Utterance_ from the user and provides a voice response - a _Dialog_.


## Examples
* "Hello"
* "How are you?"
* "What's up?"
* "Thank you"
